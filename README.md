# AI Job Application Tracker

Automated job application tracking system that extracts structured data from job descriptions using AI, reducing manual data entry time by **94%**.

---

## What This Project Does

Job seekers usually spend hours copy-pasting job details into spreadsheets.  
This project automates that entire step.

**Workflow**
1. Paste a job description into Google Sheets  
2. AI processes it automatically  
3. 13 structured fields are filled in **6–10 seconds**

No buttons. No manual triggers.

---

## Why It Matters (Data-Focused)

| Metric | Manual | Automated |
|------|--------|-----------|
| Time per job | 3–5 min | 6–10 sec |
| Time for 50 jobs | ~2.5 hrs | ~8 min |
| Time reduction | — | **94%** |
| Accuracy | Inconsistent | **95%+** |
| Manual effort | High | Zero |

---

## Extracted Fields (13)

- Company Name  
- Location (City / State / Remote)  
- Role / Position  
- Employment Type  
- Salary (Range or Hourly)  
- Experience Required  
- Key Skills  
- Extra Skills  
- Responsibilities  
- Job Summary  
- Related Projects  
- Start Term  
- Posted Date  

Missing information is handled safely using placeholders (`-`).
